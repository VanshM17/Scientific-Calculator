# Scientific Calculator

A responsive scientific calculator built using **HTML, CSS Grid, and JavaScript**.

Designed with a modern gradient UI and structured scientific layout.

---

## 🚀 Features

### 🔢 Basic Operations
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)
- Decimal support
- Clear (AC)
- Delete (⌫)

### 📐 Scientific Functions
- Powers: `x²`, `xʸ`, `10ˣ`
- Logarithms: `log`, `ln`
- Square root: `√x`
- Reciprocal: `1/x`
- Trigonometric: `sin`, `cos`, `tan`
- Factorial: `n!`
- Modulus: `mod`
- Constants: `π`, `e`
- Parentheses support `( )`

---

## 🎨 UI Highlights

- Responsive CSS Grid layout
- Structured grouping of numbers, operators, and scientific functions
- Color-coded operator column
- Highlighted `=` and `AC` buttons
- Modern gradient theme
- Button press animation

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Grid)
- Vanilla JavaScript

No external libraries used.
